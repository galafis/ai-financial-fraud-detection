Bem-vindo ao Sistema de Detecção de Fraudes Financeiras com IA

Este é um projeto de detecção de fraudes financeiras utilizando inteligência artificial, desenvolvido por Gabriel Demetrios Lafis.

O sistema utiliza um modelo de ensemble para analisar transações e identificar padrões suspeitos, ajudando a prevenir atividades fraudulentas.

Para mais detalhes sobre o projeto, incluindo código-fonte, documentação e como configurar o ambiente, visite o repositório GitHub principal.
